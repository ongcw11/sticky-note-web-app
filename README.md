# Sticky Note Web App

A simple task management web application with reminders and repeat tasks.

## Features

- Create, edit, and delete notes
- Organize tasks by categories (Work, Personal, Wishlist, Birthday, Daily)
- Set reminders for tasks
- Repeat tasks daily, weekly, or monthly
- Mark tasks as completed
- Add subtasks to notes

## Requirements

- PHP
- MySQL
- Web server (Apache/Nginx)

## Setup

1. Clone the repository
2. Create a MySQL database named `stickynote`
3. Update database credentials in `php/db_connect.php`
4. Run the application on your web server

## Usage

- Click "add new note" to create a new task
- Use filter buttons to view tasks by category
- Click the complete button to mark tasks as done
